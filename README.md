# xontrib-omx-macos

macOS utilities for Oh-My-Xonsh.

## Installation

To install use xpip:

```shell
xpip install xontrib-omx-macos
# or: xpip install -U git+https://github.com/oh-my-xonsh/xontrib-omx-macos
```

## Usage

Add this to your [rc.xsh]:

```shell
xontrib load omx_macos
```
